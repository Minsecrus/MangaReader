## ✨ 功能特性

- **📸 离线 OCR 识别**：基于 AI 模型，无需联网即可精准识别竖排日文漫画
- **✂️ 全局截图**：支持类似 QQ/微信 的截图方式，不仅限于漫画文件，可截取屏幕任意区域
- **🔍 智能分词**：自动将句子拆解为单词，标注假名读音 (Furigana)
- **🌐 多源翻译**：
  - **离线翻译**：支持加载本地翻译模型（需下载）
  - **在线 API**：支持配置 DeepL、Google、百度等 API Key 增强体验
- **🔒 隐私安全**：所有计算均在本地或直接连接官方 API，不会收集您的任何数据

## 📖 使用指南

本软件采用 **「首次配置，永久离线」** 的机制，确保核心 OCR 功能稳定可用。

### 1. 🟢 核心功能 (OCR)

_首次打开软件时程序会自动检查本地环境并下载_

- 如缺少模型，将自动连接 HuggingFace 下载到特定的项目文件夹（约 400MB），**请保持网络畅通**。
- 下载完成后，OCR 功能将**永久支持离线使用**，无需再次联网。

### 2. 📦 手动导入模型

_若自动下载失败_

如果您的网络无法连接 HuggingFace，可以手动配置：

1. 在 Release 页面下载 **「OCR 模型离线包 (ocr-model.zip)」**。
2. 打开软件设置 -> 点击 **「📂 打开模型文件夹」**。
3. 进入 `ocr` 文件夹，将压缩包内的文件解压至此（确保 `model.safetensors` 等文件直接位于该目录下）。
4. 重启软件即可。

### 3. ⚙️ 翻译与其他扩展

可以在设置中配置附加功能：

- **分词**：默认开启，辅助划分日语单词边界。
- **翻译**：**默认不包含**，如有需要请在设置中进行配置，或填入 API Key 以启用。
- **OCR快捷键**：自定义OCR快捷键。
